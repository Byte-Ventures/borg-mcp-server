Facade Ownership

borg server ... is the client facade: it forwards and renders server lifecycle evidence.
borg-mcp-server ... owns artifact verification, activation, rollback, authenticated runtime identity, and service adapters.
Neither surface may silently discover a checkout, start a daemon during setup, or claim managed persistence for a foreground process.
